# SpriteMaster - Stardew Valley Mod (SMAPI / Harmony)

Stardew Valley mod that resamples all sprites using the xBRZ resizing algorithm.

Compatible with almost any content-modifying mod.

## Requirements
* Visual Studio 2019
* Windows 7 or higher
